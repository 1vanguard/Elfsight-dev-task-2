# Elfsight Test Task (Rick and Morty API)

Application for searching and filtering characters from the Rick and Morty API.

## 🚀 Quick Start

### 1. Requirements

- Node.js ≥ 16
- npm ≥ 8

### 2. Installation

```bash
# Clone the repository
git clone https://github.com/1vanguard/Elfsight-dev-task-2.git

# Navigate to the project folder
cd Elfsight-dev-task-2

# Install dependencies
npm install
```

### 3. Run

```bash
# Start the development server
npm start
```

The application will open in your browser: http://localhost:3000
